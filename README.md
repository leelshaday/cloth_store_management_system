# cloth_store_management_system
is a software tool designed to assist cloth stores in managing their operations, inventory, and finances. This system is built using web development technologies such as HTML, CSS, JavaScript, Bootstrap, and Python's Django framework. It allows store administrators to store and manage cloth categories, product inventory, store staff and supplier information, store order data with customer information. The system provides a user-friendly interface for managing and updating inventory, tracking sales including calculating profit and loss. With Bootstrap, the system can be made mobile responsive, making it easy to use on different devices.
